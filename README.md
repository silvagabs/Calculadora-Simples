# Desafio DIO: Calculadora simples em C

##  Descrição
Calculadora simples desenvolvida em linguagem C que realiza as 4 operações básicas:
- Adição (+)
- Subtração (-)
- Multiplicação (*)
- Divisão (/)

##  Funcionalidades
- Solicita dois números ao usuário
- Permite escolher a operação desejada
- Exibe o resultado da operação
